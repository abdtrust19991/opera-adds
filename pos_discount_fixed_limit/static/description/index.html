Point of Sale Global/Cash Discount

Fixed and Percentage

  • User can enable or disable the global discount.
  • Point of Sale > Configuration > Point of Sale: Open the configuration and check 'Global Discounts'.
  • There will be a product '[Disc]Discount', Choose the product for discount.
  • User can set default fixed amount cash discount and percentage amount.
  • User will have option to apply fixed/percentage global discount from pos screen.
  • User will have freedom to apply either fixed amount global discount or percentage discount on each order. No need any configuration changes once you start the pos.
  • Option to limit discount amount or discount percentage at user level. So the POS session user can't provide more discount than configured.
