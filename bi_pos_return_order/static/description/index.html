Point Of Sale POS Product Order Return fromScreen Using Barcode Scanner Odoo Apps

This Odoo apps help to return product from specific order or return whole order using POS-Point of sales touch screen, as well as from Barcode scanner. This Odoo modules help to see the list of all the orders in Point of sales screen, user can also see individual customers list of orders also from the same screen you can able to return product of pos orders for same selected point of sales order easily using barcode scanner.After installing this Odoo module POS user can also able to see all orders on screen in POS and return the products and orders using barcode scanner with POS Order return App in Odoo.

Features

Provide Smart Button for Orders List

Provide smart button for view all pos order list an return-order on pos screen.

Display Created POS Orders List

User can see all created pos orders list on pos screen.

Order Filter Functionality on POS

User can filter order by order reference on pos screen.

Order Return Using Barcode Scanner

User can easy to return order using barcode scanner.

Smart Button for Return Order With Barcode

User can return order using pos order barcode.

Return Order Dynamically

User can return pos order dynamically by quantities on pos screen.

Print Return Order Receipt

User print receipt of return pos order.

Return POS Order Details in POS Back-end

User can see return order details in pos back-end.

Return Order With Barcode

Order Barcode in pos back-end.




In POS Screen "Return Order with Barcode" button appear, Click to Scan a Barcode for a particular product.



Scan the Barcode and Barcode will automatically read in the input field in the Pop-up. then click on apply.

Order for the scanned barcode will be read and Order lines will be added in the pop-up window.

After clicking on Apply, Products will be added to POS Order with Negative Quantity and you can place the Return Order.

After return order payment done, see payment receipt.
